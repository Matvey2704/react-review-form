# Review Form (React + TypeScript)

Небольшой проект с формой отзывов и имитацией API-запроса.

## Что реализовано

- Форма отправки отзывов
- Базовая валидация
- Loading-состояние при отправке
- Обновление списка отзывов после создания
- Имитация API-запроса

## Стек

- React
- TypeScript
- Vite
- CSS

## Как запустить

```bash
npm install
npm run dev
